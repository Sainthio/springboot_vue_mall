## 🎯 核心要点（必须掌握）

### 必须会回答的5个问题

#### 1️⃣ 为什么选择Spring Boot？
**回答**：快速开发、自动配置、内嵌Tomcat、生态完善

#### 2️⃣ 如何防止超卖？
**回答**：数据库行锁 + WHERE条件判断库存
```sql
UPDATE shangpinxinxi 
SET alllimittimes = alllimittimes - 1
WHERE id = ? AND alllimittimes >= 1
```

#### 3️⃣ 系统有哪些核心功能？
**回答**：
- 用户端：商品浏览、购物车、订单管理
- 管理端：商品管理、订单管理、数据统计

#### 4️⃣ 前后端如何交互？
**回答**：RESTful API + AJAX + JSON

#### 5️⃣ 系统的并发量是多少？
**回答**：支持1000并发，平均响应时间500ms
